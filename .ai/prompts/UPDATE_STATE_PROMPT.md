# Prompt: Update State

Purpose: update project state after a human or AI-made change.

## Instructions

1. Read `.ai/AI_CONTRACT.md`.
2. Read `.ai/STATE.md`.
3. Read the relevant phase, slice, ADR, or verification files.
4. Update current phase and current slice.
5. Record completed work.
6. Record decisions or link to ADRs.
7. Record known risks.
8. Record open questions.
9. Record validation status.
10. Keep state concise and factual.

## Output

- Updated `.ai/STATE.md`.
- Any linked phase, slice, ADR, or verification updates.
- Remaining unknowns.
